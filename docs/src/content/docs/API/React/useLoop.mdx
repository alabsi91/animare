A custom react hook to make it easy to create game loops in React.

```ts wrap title="syntax"
function useLoop(onUpdateCallback: (delta: number) => void, deps: React.DependencyList = []): void;
```

Based on [Loop](/animare/api/animare/loop)

## Usage

```ts title="example" frame="terminal"
import { useLoop } from 'animare/react';

function MyComponent() {
  useLoop(delta => {
    // do something
  }, []);
}
```
