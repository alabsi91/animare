Create a game loop.

```ts title="syntax"
function loop(onUpdateCallback: (delta: number) => void): () => void;
```

## Usage

```ts title="example" frame="terminal"
import animare from 'animare';

const stop = animare.loop(delta => {
  // do something
});

stop(); // stop the loop if needed
```

To use with React see: [useLoop](/api/react/useloop)
