For determining the position of the animation in the timeline.

### AfterPrevious

`AnimationTiming.AfterPrevious`

Play the animation after the previous one finishes.

### WithPrevious

`AnimationTiming.WithPrevious`

Play the animation simultaneously with the previous one.

### FromStart

`AnimationTiming.FromStart`

Play the animation from the start of the timeline.
