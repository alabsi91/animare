animare comes with a few predefined easing functions.

```ts title="example" frame="terminal"
import { ease } from 'animare/plugins';

animare.single({ to: 100, ease: ease.out.quad }, info => console.log(info.value));
```

## Wobble()

`ease.in.wobble(bounciness = 1)` <Badge text="in" variant="caution" size="small" /> <br />
`ease.out.wobble(bounciness = 1)` <Badge text="out" variant="caution" size="small" /> <br />
`ease.inOut.wobble(bounciness = 1)` <Badge text="in-out" variant="caution" size="small" />

Creates a simple elastic interaction, similar to a spring oscillating back and forth.
The default bounciness is `1`, which overshoots a little bit once.
A bounciness of `0` doesn't overshoot at all,
and a bounciness of `N > 1` will overshoot about `N` times.

<Ease title='in.wobble(0)' padding={20} easing='ease.in.wobble(0)' client:idle />
<Ease title='out.wobble(1)' padding={20} easing='ease.out.wobble(1)' client:idle />
<Ease title='inOut.wobble(6)' padding={20} easing='ease.inOut.wobble(6)' client:idle />

`ease.steps(steps = 10, start = true)`

Creates a staircase easing function.

`steps` The number of steps. <br/>

## FromPoints()

`fromPoints(values: Float32List)`

Creates a custom easing function from a pre-generated array of points using [Animare Ease Visualizer](https://animare-ease-visualizer.netlify.app/).

This could be useful for complicated custom easing functions that can effect the frame rate of the animation.

Use the [Animare Ease Visualizer](https://animare-ease-visualizer.netlify.app/) tool to create the easing function.
