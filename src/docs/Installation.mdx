## Installation

Install using your package manager of choice to add animare to your project:

<Tabs>
    <Item label='NPM' default>
        <br/>
        ```apache title="NPM"
        npm install animare
        ```
    </Item>

    <Item label='Yarn'>
        <br/>
        ```apache title="Yarn"
        yarn add animare
        ```
    </Item>

</Tabs>

Aaaaand, you’re done! That was fast. 🐇
